NPxP

Next generation plugin of Piece by Piece

Download this project as a .zip file Download this project as a tar.gz file

Requirement

Windows XP/Vista/7 .NET Framework 3.5

Installation

Install DevExpress dll

Move dlls into C:\Windows\assembly. Notice: it has 8 dlls. That's map chart control.

Deploy Parameter Files::NPxP Config

Copy all folder to C:\Program Files\Web Ranger\Parameter Files\ Those config might not appropriate for your project, but next step will help you setup this. You just copy that. Check the NPxP folder has system,grade,map folders.

Install plugin

PlugIns has three folders, that vaersion 2.7.3 , 2.8.x+, Others

2.7.3

only support Web Inspector 2.7.3. that is not append xml of config into database. but fixed memory issue, and other bug issue.

2.8.x+

It's fixed current all issue. but not test with machine in production environment.

Others

The folder has some dll that for backup. System occur crush in XP sometime that cause RT and NPxP config broke. If you encounter this situation please check all config files.(Parameter Files) and use dlls.

Notes

In 2.7.3 version necessary a C:\temp folder that debug tool for log of exception. If you can't open software that show all empty on map please check that folder exists.

Launch

When you first start Web Inspector with NPxP Plugin . You should set some config in debug mode. That press Ctrl + d to open debug mode. and turn off the default Flawmap. That's done. You can see screen as follow :

Quick Setup

Switch tab to Queue and load your job (mcs). then click Map Setting button to open window. Click save. that's done.

This step for load your defect data into map config.

Usage::Detail

Map Setting

The map setting is for config map ex: shape, color.

Start Web Inspector and switch tab to Queue Click Edit job to config for NPxP.

The NPxP Plugin need some piece data from this. So it checked and enter width and height of Piece.

Click Load Next job that help NPxP Plugin get defect and piece data.

Click Map Setting open the setup window, You can setup your preferences here for map. If you first time use NPxP Plugin, even if doesn't edit anything , you should click save .

Double click on cell then open color setup dialog.

Check your preferences and save then should see success message.

The legend can help you hide or show defect on map

The shape is show your preferences.

Grade Setting

The Grade is a custom scoring mechanism.

ROI Setting

The first step you should config grid's columns and rows. Then click Create

Set each grid size. Notice your unit that follow job.

Grade Setting

Switch to Grade Setting Tab and sub tab Point is mean your defect. You can set each defect score.

Notice each grid has score own. If you want to set once for all grid(sub piece). Please select All and checked Same value from [All] tag

Each grid(sub piece) has a score that can get a level. According this level and score list. You can set A-E and assign a score. example: A:10 B:20 when grid get 15 that level is A. Other score belongs to F(ail).

Pass or Fail Score is define a piece will send what event. A-F Again according this list, All grid(sub piece)'s total score will get a level (Event). Now all config is done.

Start Inspect

Click Start to start inspect.

Top area will show about pass number , fail number, yield, and Total Score of this piece.

Click Fail Piece List to open window that show all fail piece

Click the row will switch to that fail piece

The legend list help you show just you want to see.

Click the right table's tab will selection a defect and GridView selection realtime.

You can change pieces by Prev or Next button.

Double click will show the defect detail information. You can Zoom in out this image of defect.
